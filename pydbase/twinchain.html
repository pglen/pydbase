# Twinchain
 
Block chain layer on top of twincore.
 
    prev     curr
        record
|   Time Now    |   Time  Now    |  Time Now     |
|   hash256   | |    hash256   | |   hash256   | |
|   Header    | |    Header    | |   Header    | |
|   Payload   | |    Payload   | |   Payload   | |
|   Backlink  | |    Backlink  | |   Backlink  | |
              |----->---|      |---->---|     |------ ...
 
The sum of fields saved to the next backlink.
